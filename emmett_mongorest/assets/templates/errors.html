Errors

{{ =title }} uses conventional HTTP response codes to indicate the success or failure of an API request. In general: Codes in the 2xx range indicate success. Codes in the 4xx range indicate an error that failed given the information provided (e.g., a required parameter was omitted, a charge failed, etc.). Codes in the 5xx range indicate an error with API's servers.

Some 4xx errors that could be handled programmatically include an errors attribute in response which includes details about the error reported.
