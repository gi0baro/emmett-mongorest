Pagination

API resources might have support for bulk fetches via "list" API methods. These list API methods share a common structure, taking at these three parameters: page, page_size, and sort_by.

Parameters

  • page optional, default is 1
  • page_size optional, default is {{ =page_size }}
  • sort_by optional

Returns

Returns an object with the following attributes:
  • meta mapping[string, any]
    • object string
    • total_objects integer
    • has_more boolean
  • data list[any] 
                
{
  "meta": {
    "object": "list",
    "total_objects": 0,
    "has_more": false
  },
  "data": []
}
